# Contributing

## Environment Setup

Ensure you have the following tools installed
 - `node`
 - `npm`

Install the project dependencies

```sh
npm install
```

## Running a Development Server

```sh
npm run dev
```

## Building for Production

```sh
npm run build
```

The built website will be at `./dist/web`
